# Excelsuperpower

Excel优化打印小程序。

## MVP定位

Excelsuperpower 是一个面向办公打印场景的小程序工具：用户上传 Excel 文件后，系统自动识别 Sheet、诊断打印问题、推荐打印优化方案，并生成适合打印的 Excel / PDF 文件。

第一版只聚焦 Excel 打印优化，不同时扩展 Word、复杂模板库和企业协作。

## MVP核心流程

1. 上传 `.xlsx` / `.xlsm` 文件。
2. 自动识别工作簿中的 Sheet。
3. 用户选择需要打印的 Sheet。
4. 系统给出 3 类推荐方案：
   - 一页宽打印：适合台账、清单、明细表。
   - 横向分页打印：适合列很多的宽表。
   - 批量 Sheet 打印：适合多个报表连续打印。
5. 用户确认方案后生成：
   - 可打印 PDF。
   - 优化后的 Excel 文件。
   - 高级版 `.xlsm` 批量打印按钮文件。

## 产品边界

MVP必须做：

- Sheet 识别与多选。
- 打印区域自动识别。
- 横向 / 纵向推荐。
- 首行重复打印建议。
- 一页宽适配建议。
- 结果文件下载入口。

MVP暂不做：

- Word 文档排版。
- 完整在线预览渲染。
- 企业多人协作。
- 复杂模板市场。
- 小程序内直接执行 VBA。

## 技术方向

- 小程序端：微信小程序原生页面。
- 后端：文件上传、任务状态、结果下载接口。
- Excel处理：Python `openpyxl` / LibreOffice headless。
- AI模块：负责诊断、推荐和解释，不直接修改文件。
- 排版执行：由确定性规则引擎完成。
