# Excelsuperpower MVP设计

## 一句话定位

上传 Excel，自动诊断打印问题，推荐打印布局，一键生成适合打印的文件。

## 目标用户

- 行政、人事、财务、销售内勤。
- 学校老师、培训机构运营人员。
- 打印店和经常替别人整理表格的人。

## 核心痛点

Excel 打印前的设置很分散，用户经常遇到：

- 列太多导致内容被切到多页。
- 表头没有每页重复。
- 打印区域包含无效空白。
- 多个 Sheet 需要逐个设置。
- 横向、纵向、缩放比例难判断。
- 生成 PDF 后版式才发现不对。

## MVP功能清单

### 1. 文件上传

- 支持 `.xlsx`。
- 预留 `.xlsm`，用于未来保留宏或生成批量打印按钮。
- 限制文件大小，建议 MVP 先设为 20MB。

### 2. Sheet识别

后端解析工作簿，返回：

- Sheet 名称。
- 数据行列范围。
- 是否隐藏。
- 是否可能为空表。
- 是否包含合并单元格。
- 初步推荐打印方向。

### 3. 打印诊断

为每个 Sheet 生成诊断项：

- 打印区域是否过大。
- 是否需要重复标题行。
- 是否适合横向打印。
- 是否需要一页宽。
- 是否存在过窄字体风险。
- 是否存在大量空白行列。

### 4. 推荐方案

MVP固定提供三种方案：

- 标准清晰版：保留字号，设置打印区域和重复标题行，适合正式打印。
- 节纸紧凑版：缩小边距，适配一页宽，适合内部流转。
- 宽表横向版：横向打印，按宽表分页，适合列很多的表。

### 5. 文件生成

MVP输出：

- 优化后的 `.xlsx`。
- 打印稳定的 `.pdf`。

高级能力预留：

- `.xlsm` 批量打印按钮。
- 按 Sheet 分 PDF。
- 按分类字段拆分打印。

## 用户流程

1. 用户进入首页。
2. 点击上传 Excel。
3. 系统显示识别到的 Sheet 列表。
4. 用户选择要处理的 Sheet。
5. 系统展示推荐方案和诊断摘要。
6. 用户选择方案并点击生成。
7. 系统展示生成结果，用户下载 PDF 或 Excel。

## AI职责边界

AI负责：

- 解释为什么推荐某个方案。
- 根据表格结构选择策略。
- 给用户生成易懂的诊断摘要。

确定性代码负责：

- 设置打印区域。
- 设置纸张方向。
- 设置页边距。
- 设置重复标题行。
- 设置缩放比例。
- 生成 PDF 和 Excel 文件。

## 商业化建议

免费版：

- 单文件处理。
- 输出 PDF。
- 每日有限次数。

付费版：

- 批量文件处理。
- 输出优化 Excel。
- 输出带批量打印按钮的 `.xlsm`。
- 更大文件。
- 企业模板和历史记录。
